Frost Legion Demonlist

Welcome to the Geometry Dash Demonlist tracker. Use this page to view demons, submit new demons, and find team rules. This is a single-page layout inspired by your Google Sites design.

Submit a Demon

Use this form as a template for your own Google Forms or GitHub Issue link. This site is static, so we show the details you should gather: Demon name, difficulty, creator, reason.

  • Required: Demon name
  • Required: Difficulty (Insane Demon, Extreme Demon, etc.)
  • Required: Creator + verification link
  • Optional: Notes from team check

About Frost Legion

Frost Legion Demonlist is a community-run Geometry Dash list. We gather demon levels and rank them by team consensus. This page is built as a static GitHub Pages site, with HTML/CSS/JS only.

How this works:

  • We keep a list of candidate demons.
  • Team members test and review each demon.
  • Approved demons stay on the list until rotated.
